# Uber ride Analysis for the year 2024

This project analyse the ride trend, hotspot locations and rating based analysis

## Dataset Description
- Date
- BookingID
- Booking Status
- CustomerID
- Vehicle type
- Pickup location
- Drop location
- Reason for cancelling
- Driver cancellation reason
- Incomplete ride reason
- booking value
- ride distance
- driver rating
- customer rating
- payment method
